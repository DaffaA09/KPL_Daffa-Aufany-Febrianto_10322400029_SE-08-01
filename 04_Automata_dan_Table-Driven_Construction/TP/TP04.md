```CSS
.mode-gelap {
    background-color: #171b25;
    color: #EBECF7;
}

.mode-gelap .kotak-input{
    background-color: #2e3443;
    color: #EBECF7;
}

.mode-gelap button{
    background-color: #29ddcc;
    border: none;
}
