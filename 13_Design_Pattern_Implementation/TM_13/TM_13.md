## Deskripsi
Event delegation ialah sebuah teknik dalam JavaScript untuk menangani event dengan cara memasang satu event listener pada elemen parent, bukan pada setiap elemen child satu per satu. Teknik ini memanfaatkan konsep event bubbling, yaitu ketika event dari elemen child naik ke elemen parent.

Dengan event delegation, kode menjadi lebih efisien, rapi, dan mudah dikelola, terutama jika ada banyak elemen atau elemen dibuat secara dinamis. Contohnya, pada daftar tombol hapus di todo list, kita cukup memasang listener pada parent ul, lalu mengecek tombol mana yang diklik menggunakan event.target.

singkatnya , event delegation : parent menangani event yang terjadi pada child.
